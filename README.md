#### How to run dApp

To run stand-alone:

```
git clone https://github.com/nfnth/nfnth
python nfnth/site.py
```

To install and run, using Arch Linux on [Linode](https://linode.com):

```
pacman -S git
git clone https://github.com/nfnth/nfnth
chmod +x nfnth/install.sh
./nfnth/install.sh
```

To run client:

```
wget ocuros.iso
dd if=ocuros.iso of=/dev/sda status=progress
```
